# Simple Interest Calculator

This project calculates Simple Interest using the formula:

SI = (P × R × T) / 100

Where:
P = Principal amount
R = Rate of interest
T = Time in years

Example:
If P = 1000, R = 5%, T = 2 years

SI = (1000 × 5 × 2) / 100
SI = 100
